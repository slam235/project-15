## Сборка и запуск
- Склонируйте репозиторий:

`git clone https://github.com/slam235/project-15.git`

- Установите необходимые зависимости:

`npm install`

- Для запуска с hot reload выполните команду:

`npm run dev`

- Для запуска без hot reload выполните команду:

`npm run start`
